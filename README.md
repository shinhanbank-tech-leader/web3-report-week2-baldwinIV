# 제목: 비트코인 결제와 기존 금융 지급결제의 구조적 차이 비교

## 레포트 내용

이번 주에 학습한 UTXO 모델


### 1. 비트코인 결제 시스템의 핵심: UTXO 모델 (+ 단점 그리고 이더리움의 계정 기반 모델까지..)

비트코인 네트워크의 결제 방식은 일반적인 은행 계좌의 잔액 개념과는 근본적으로 다릅니다. 비트코인은 미사용 출력값인 UTXO(Unspent Transaction Output) 방식을 채택하고 있습니다. 이는 사용자가 특정 금액의 잔고를 보유하는 것이 아니라, 아직 소비되지 않은 이전 거래의 결과물들을 보유하고 있음을 의미합니다.

결제가 발생하면 사용자는 자신이 가진 UTXO들을 입력값으로 사용하며, 이를 통해 새로운 UTXO를 생성하여 수취인에게 전달합니다. 이 과정에서 송금액을 제외한 나머지 금액은 본인의 새로운 주소로 반환되는 잔돈 UTXO로 생성됩니다. 모든 거래는 개별적인 데이터 조각들의 조합과 분해로 이루어지며, 이를 통해 이중 지불을 방지하고 거래의 투명성을 확보합니다.

*** 
다만 UTXO 모델은 **개발의 복잡성, 확장성의 한계, 무상태성** 이라는 이슈가 있습니다. 네트워크의 부담이 커지고, 특히 상태를 가질 수 없기 때문에. 스마트 컨트랙트 개발에 부적합하다고 합니다. 
이 내용은 기존 금융결제 시스템과 비트코인 결제 시스템의 차이를 비교하기에는 주제가 다르니 여기까지만 하겠습니다.


### 2. 기존 금융 지급결제 시스템의 구조
기존의 금융 시스템은 중앙 집중형 장부 기록 방식을 기반으로 합니다. 사용자가 결제를 요청하면 은행과 같은 신뢰할 수 있는 제3의 중개 기관이 각 계좌의 잔액을 확인하고 수정합니다.

이 구조는 크게 발행 기관, 매입 기관, 그리고 중앙의 청산 결제 기관으로 구성됩니다. 예를 들어 신용카드 결제 시 가맹점, 매입사, 카드 네트워크사, 발급사 간의 복잡한 정보 교환이 일어납니다. 최종적인 자금의 이동은 실제 거래 시점보다 늦게 발생하며, 중개 기관들이 장부상의 숫자를 조정함으로써 결제가 완료되는 방식입니다. 이 과정에서 각 단계의 중개인은 거래의 유효성을 검증하고 수수료를 취득합니다.

### 3. 기존 금융 시스템 대비 비트코인 결제의 장단점
비트코인 결제는 기존 시스템과 비교했을 때 뚜렷한 명암을 가지고 있습니다.

장점 측면에서는 중개 기관이 없으므로 국경 간 송금 시 복잡한 환전 절차나 중개 수수료를 대폭 절감할 수 있습니다. 또한 24시간 중단 없이 운영되며, 검열 저항성을 가집니다. 특정 기관이 거래를 임의로 중단하거나 거부할 수 없다는 점이 강력한 특징입니다.

 * UTXO모델의 장점도 있는데요.  UTXO 모델은 각 거래의 결과물을 독립적인 데이터 조각으로 관리합니다. 특정 비트코인을 사용하려면 반드시 이전 거래에서 생성된 미사용 출력값을 참조해야 하며 이를 사용함과 동시에 해당 데이터는 즉시 소비된 상태로 변경됩니다.(spent) 이러한 구조는 동일한 자산을 두 번 사용하는 이중 지불 시도가 원천적으로 차단됩니다.


반면 단점으로는 확장성과 변동성이 꼽힙니다. 초당 처리할 수 있는 거래 건수(TPS)가 기존 결제 네트워크에 비해 현저히 낮아 실시간 소액 결제에 부적합합니다. 또한 거래 확정까지 일정 시간이 소요되는 확정 지연 문제가 존재하며, 가치 변동성이 커서 결제 수단으로서의 안정성이 부족합니다.

### 4. 비트코인 결제 단점의 보완 방안
비트코인의 기술적 한계를 극복하기 위해 다양한 레이어 2 솔루션이 개발되고 있습니다. 가장 대표적인 것은 라이트닝 네트워크입니다. 이는 개별 거래를 매번 메인 블록체인에 기록하지 않고, 별도의 오프체인 채널을 통해 즉각적이고 저렴한 비용으로 다수의 거래를 처리한 뒤 최종 결과만 블록체인에 기록하는 방식입니다. 이를 통해 확장성 문제를 해결할 수 있습니다.

또한 가치 변동성 문제를 해결하기 위해 비트코인을 담보로 하거나 알고리즘을 통해 가격을 일정하게 유지하는 스테이블코인과의 연동 연구도 활발합니다. 제도권 내에서는 수탁 서비스(Custody)를 통해 거래의 안전성을 높이고, 법적 규제 프레임워크를 구축하여 사용자 보호 기능을 강화하는 방향으로 보완이 이루어지고 있습니다.
   
   - 이더리움의 스마트 컨트랙트 위에 비트코인을 ERC-20 토큰화 하여 거래할 수도 있습니다. (WBTC - wrapped bitcoin)
